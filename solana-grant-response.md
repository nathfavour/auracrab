# Auracrab — Superteam Agentic Engineering Grant

Auracrab is a Solana-native agentic OS — “OpenClaw, but built natively for Solana.” Agents like OpenClaw are impressive, but they lack first-class support for the Web3 ecosystem, especially Solana, and Auracrab fixes that with day-one support for agents, agentic wallets, x402, onchain skills, and Solana-native workflows.

solana.new is the right environment for this because it already centers AI-assisted building with Claude/Codex, MCPs, skills, and fast crypto shipping. The grant would help us extend Auracrab from a general agent platform into a crypto-native execution layer for Solana builders.

With this grant, I would ship wallet tooling, Solana MCP connectors, onchain task skills, and an end-to-end demo showing an agent planning, executing, and reporting on Solana-native work. The goal is to make Auracrab a real alternative for builders who want agentic engineering that is actually native to crypto.
