# 🦀 AURACRAB: THE BIOLOGICAL SYSTEMS ROADMAP

This roadmap tracks the evolution of Auracrab into a self-sustaining, resilient, and autonomous biological entity.

## Phase 1: Proprioception (Internal Sense)
*Goal: The system must know itself to manage itself.*
- [ ] **Internal State Injection**: Feed `Energy` levels (CPU/RAM) and `Node Health` directly into the System Prompt.
- [ ] **Metabolism Tracking**: Implement an "Energy Cost" for every API call and compute-intensive action.
- [ ] **Thermodynamic Reasoning**: Enable the AI to postpone or abort tasks based on energy thresholds.

## Phase 2: The Task Nervous System (Asynchronous Granularity)
*Goal: Move from monolithic replies to a recursive, pulsed execution model.*
- [ ] **Recursive Task Tree**: Break high-level user goals into atomic, schedulable `Pulse` actions.
- [ ] **Spine-Integrated Execution**: Tasks no longer block; they attach to the Spine and execute over multiple heartbeats.
- [ ] **Progress Pulsing**: Automatically drop "Lazy I/O" updates to the operator as sub-tasks complete.

## Phase 3: Communication Bands (I/O Architecture)
*Goal: Specialized pathways for different priorities.*
- [ ] **Fast I/O Band**: Immediate, high-priority interrupts for operator commands.
- [ ] **Lazy I/O Band**: Throttled background state synchronization and status updates.
- [ ] **Isolated I/O Band**: Secure, encrypted node-to-node channels for swarm coordination.

## Phase 4: Metabolism & Hibernation
*Goal: Resource efficiency and systemic longevity.*
- [ ] **Metabolic Substrate**: Store "Path of Least Resistance" in memory to avoid redundant high-energy re-thinking.
- [ ] **Hibernation Mode**: Slow the Heartbeat (Spine) to 0.1Hz when idle to minimize VPS overhead.
- [ ] **Automated Apoptosis**: Systemic cleanup of "dead cells" (temp files, stale clones, old logs) during idle cycles.

## Phase 5: Swarm Consensus (Distributed Immune System)
*Goal: Collective intelligence and zero-touch resilience.*
- [ ] **Node Hand-off**: Ability for an overloaded node to delegate metabolic load to a peer.
- [ ] **Consensus Voting**: Swarm-wide voting to terminate "mutated" or runaway processes.
- [ ] **Autonomous Cloning**: Self-replication triggered by high systemic demand and healthy energy reserves.
